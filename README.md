# UniswapV3 Swap Bot

## Description

This bot detects transactions with UniswapV3 swaps

## Supported Chains
- Ethereum

## Alerts

- UNI-1
  - Fired when a transaction contains a UniswapV3 swap
  - Severity is always set to "low" 
  - Type is always set to "info"
